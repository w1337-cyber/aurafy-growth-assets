AFTER — proposed

Interactive block (above): live seat counter, single primary CTA, graceful sold-out state, UTM-tagged for attribution.

What's needed to ship this

  • Phase 1 (HTML only — ships today): drop the block into /pricing + homepage + in-app upgrade modal. Counter falls back to "<50" until the endpoint exists. Already 80% of the lift.
  • Phase 2 (5-line endpoint): GET /api/founder-seats → {"left": N}, cached 60s. Counts active $19 subscriptions in the existing billing layer.
  • Phase 3 (telemetry): data-track="founder-cta" hook for future analytics; UTM attribution to count converted visitors in Stripe.
