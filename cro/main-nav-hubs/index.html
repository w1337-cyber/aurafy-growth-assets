Why these 3 hubs

/guides/

4 broker export how-tos (Tradovate, NT8, Sierra, Rithmic) + cross-platform cleanup framework. Action-intent traffic.

/prop-firms/

7 firm landings (Apex, Topstep, MFFU, TPT, Tradeify, Bulenox, Earn2Trade) + 3-rule-axes comparison table. Research-intent traffic.

/brokers/

4 broker landings (Tradovate, NinjaTrader 8, Sierra Chart, Rithmic R|Trader Pro) + routing decision-blocks. Pre-qualified-platform traffic.

1

Tools + Guides pair. Both are action-intent (calculators help you decide; guides help you act). Slotting them adjacent in nav creates a visible action-side cluster.

2

Prop Firms + Brokers pair. Both are research-intent (firm rules + execution-platform comparisons). Adjacent placement makes the IA legible.

3

Blog stays last. Loosest topical fit — magazine-style content vs structured hubs.
