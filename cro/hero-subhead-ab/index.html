Cookie-based sticky assignment

A first-time visitor gets randomly assigned to A/B/C with equal probability. The choice is stored in a 30-day cookie so the same visitor sees the same subhead across return visits. What we'd measure (decision matrix)

MetricWhat it tells us
Hero CTA click rate (per variant) Did the subhead make them want to start free? Immediate but noisy.
Hero → free signup (per variant) The real number. Did clicks turn into actual accounts?
Free signup → paid (per variant) The most important number. A subhead that pulls in visitors who never pay = wasted lift.
Bounce rate on / (per variant) Did the subhead push undecideds away? (Variant A may do this.)
Variant × traffic source Reddit vs tools vs organic — which variant wins for which channel?

What's needed to ship this

  • Pre-req: CRO #002 (hero CTA flip) should ship first or simultaneously — Variant A and B both assume the CTA is "Start free — no card." If still "See pricing," only Variant A reads coherently.
  • Snippet (~30 lines): drop into the hero — assigns a variant, sets a 30-day cookie, swaps the subhead text, appends utm_content=subhead-A|B|C to every hero CTA.
  • Impression event: if aurafy.dev has analytics (PostHog / GA4 / Plausible), the snippet fires one hero_subhead_shown event per session so denominators are clean.
  • Decision: after 2–4 weeks or ~500 visitors/variant, pick the winner based on free→paid (not CTR). Roll out winner to 100%. Variant copy can be reused in retargeting + email subject lines.
  • Reversible: remove the script block; the subhead element shows whatever default is in the HTML. 60-second rollback.

What stays inside the hard rules

  • No fake claims. All three subheads are factually verified — TradeZella $49 + Loom $15 + FX Replay $39 = $103 is real; rule violations are the #1 prop-firm churn cause; Aurafy genuinely is the only major futures journal with native screen recording (verified in our competitor cluster).
  • No dark patterns. No fake urgency, no fake testimonials, no popups, no exit-intent. Just three legitimate value props, randomly served.
  • No identity manipulation. No fake user reviews tied to a variant, no manipulated social proof.
  • Honest random. 1/3 each, no weighting — we're not pre-judging the winner, that defeats the test.
